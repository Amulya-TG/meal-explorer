### 🍽 Meal Explorer

A React.js web application that allows users to explore different meal categories, view dishes under each category, and read the complete recipe for any selected dish.


### 🚀 Features

- ✨ View different meal categories
- 🍛 Explore dishes under each category
- 📄 View detailed recipe in popup modal
- 🔎 Search meals by name or category
- ⚡ Fast and responsive UI
- ❤️ Add / remove favorites (stored in localStorage)
- 🌙 Dark / Light mode toggle

🛠️ Technologies Used

- Frontend: React.js
- Routing: React Router DOM
- State Management: React Hooks (useState, useEffect)
- API: TheMealDB API
- Styling: CSS

### 📌 How It Works

- Home Page
    → Welcome screen
- Meal Page
    → Displays all meal categories
- Meals Page
    → Shows meals based on selected category
- Search
    → Displays filtered meals
- Recipe Popup
    → Shows full cooking instructions

### ⚙️ Installation

- Clone the repository:
git clone https://github.com/Amulya-TG/Meal-Explorer.git

- Navigate to the project folder:
cd meal-recipe-explorer

- Install dependencies:
npm install

- Start the development server:
npm run dev
